# T088 — Require a confirmed Last.fm match to add an album review

## Do

Today, on `src/pages/reviews/albums/add.js`, the Last.fm lookup match (`lastfmMatch` state) is OPTIONAL
— the form can be submitted whether or not the user ever ran/confirmed a search, since the match
fields are only conditionally spread into the payload (`...(lastfmMatch && { lastfm: lastfmMatch.lastfm , ... })`).
Make confirming a Last.fm match REQUIRED to submit this form:

1. **Client-side**: disable the submit button (or block `handleSubmit`) until `lastfmMatch`
   is non-null (i.e. the user has run a search and picked a candidate). Show a clear inline message
   near the search/match UI when no match is selected yet (e.g. "Search and select a Last.fm
   match before saving").
2. **Server-side**: in `src/pages/api/reviews/albums/add.js`, add `lastfm` to the required-fields check — reject
   the request with HTTP 400 (matching this route's existing error-response style) if
   `lastfm` is missing from the request body, alongside the existing required-field
   checks (title/rating/etc.) already enforced there.

Do NOT touch the edit/update flow (`src/pages/reviews/albums/edit/[id].js` or the
corresponding `update.js` API route) — this idea is scoped to the ADD flow only, per the
original request ("when adding a new [type]").

## Done when

- Submitting `src/pages/reviews/albums/add.js`'s form without ever having selected a Last.fm match is blocked
  client-side (the submit action does not fire / is disabled), with a visible message explaining
  why.
- `POST /api/reviews/albums/add` (or the route this file
  serves) rejects a request missing `lastfm` with HTTP 400, even if all other required
  fields (title, rating, etc.) are present — a test (new or added to an existing test file for
  this route, if one exists) asserts this.
- Submitting with a confirmed match (i.e. `lastfm` present) still succeeds exactly as
  before — no regression to the existing happy path.
- The edit/update flow for this review type is unchanged — this task does not touch it.
