## Overview
The Now Playing widget (src/components/NowPlaying.js, rendered in the Header on every page and on the home page) shows a pill (.now-playing-chip) when a track is playing. Right now only the scrolling marquee text is a clickable Last.fm link — the status dot, the pill padding, and the rest of the pill are dead zones, even though the whole thing looks tappable. Make the ENTIRE pill a single Last.fm link when a track is playing AND a Last.fm URL is available, so clicking anywhere on it (dot included) opens the Last.fm URL. This is purely a hit-target / quality-of-life improvement; the pill must look the same aside from being fully clickable. It only applies to the playing-with-URL case: the loading chip, the offline 'not listening rn :)' chip, and the playing-but-no-URL case all stay non-clickable.
