# T017 — Obtain a TMDB API key (human step)

## Do

Create a TMDB (themoviedb.org) API key and make it available to the app as the
`TMDB_API_KEY` environment variable. The autonomous loop can't sign up for a third-party
account, so this is a human step.

- Register for a TMDB API key.
- Add `TMDB_API_KEY` to Vercel (production) and to local `.env.local`
  (`vercel env add TMDB_API_KEY production`).

## Done when

- `TMDB_API_KEY` is set in Vercel and in local `.env.local`, and a sanity test (e.g. a
  curl to the TMDB search endpoint with the key) returns results — so the search route
  (T018) has a working key to use.
