# T017 — Make the TMDB API token available (human step)

## Do

Make a TMDB (themoviedb.org) credential available to the app as the `TMDB_API_TOKEN`
environment variable. This is TMDB's **v4 "Read Access Token"** (a JWT, used as an
`Authorization: Bearer <token>` header), NOT the older v3 `api_key` query param.

A working token already existed in another local project of the owner's and has been copied
into this repo's gitignored `.env.local` as `TMDB_API_TOKEN`. The remaining step (human,
needs Vercel auth) is to add it to Vercel so the deployed site has it:

```
vercel env add TMDB_API_TOKEN production
# paste the same token value when prompted
```

## Done when

- `TMDB_API_TOKEN` is set in Vercel (production) AND present in local `.env.local`
  (already done locally), and a sanity check (a TMDB search call with
  `Authorization: Bearer $TMDB_API_TOKEN`) returns results — so the search route (T018)
  has a working credential in every environment.
