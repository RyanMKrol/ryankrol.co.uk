# T070 — Backfill-from-API button on the movies edit page

## Do

On `src/pages/reviews/movies/edit/[id].js`, add a "Backfill from TMDB" button using the shared
`MetadataBackfillModal` component from T066. Wire it so:
- The search is seeded from this form's current `title` field, calling
  `GET /api/tmdb/search?query=...&type=movie` (see `src/pages/api/tmdb/search.js`, which returns
  an array of `mapTmdbResult` shapes: `{ tmdbId, mediaType, title, overview, posterPath, date }` —
  see `src/lib/tmdb.js`).
- On confirm, set this form's `tmdbId`, `mediaType`, `posterPath`, `tmdbOverview` (map the
  result's `overview` field to `tmdbOverview`, NOT to this form's own `gist` field — see the
  warning below), and `tmdbDate` (map the result's `date` field) fields, per the "overwrite all
  metadata fields" behavior T067 now supports on `POST /api/reviews/movies/update`.
- **CRITICAL naming trap: the TMDB result's `overview` field (TMDB's own synopsis text) is
  COMPLETELY DIFFERENT from this form's `gist` field (the user's own written review).** Map TMDB's
  `overview` → this record's `tmdbOverview` field ONLY. Never write to `gist` from this modal, and
  never touch `rating`.
- The existing Save/Delete flow for this edit page is otherwise unchanged — the backfill button
  just pre-fills form fields; the user still clicks the page's normal Save button afterward to
  persist (do not auto-submit on confirm).

## Done when

- Loading `/reviews/movies/edit/[id]` for an existing movie review, clicking "Backfill from TMDB",
  picking a search result, and confirming updates the form's `tmdbId`/`mediaType`/`posterPath`/
  `tmdbOverview`/`tmdbDate` fields visibly (e.g. a poster image appears/updates), without touching
  `rating` or `gist`.
- Clicking the page's existing Save button afterward persists the backfilled fields via
  `POST /api/reviews/movies/update` (confirm by reloading the edit page and seeing the new values
  still present).
- No file outside `src/pages/reviews/movies/edit/[id].js` and `src/components/**` was modified.
