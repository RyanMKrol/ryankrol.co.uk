## Do

Run the create-table script from the prior task to provision the `HotTakes` table in DynamoDB. This requires the owner's AWS credentials, so it cannot be done by the autonomous loop.

```
npm run hot-takes:create-table
```

## Done when

`npm run hot-takes:create-table` has completed successfully against the owner's AWS account, and the `HotTakes` table shows status ACTIVE in DynamoDB (region `us-east-2`) with a single String partition key `id`.
