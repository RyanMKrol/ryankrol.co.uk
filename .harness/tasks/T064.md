# T064 — Review the three complete UI rewrite variants and pick a winner

## Do

All three variants (Variant 1 at `/v1`, Variant 2 at `/v2`, Variant 3 at `/v3`) are now fully
built, per the design briefs in `.harness/designs/T042-ui-rewrite-briefs.md`. Run the dev server
(`npm run dev`) and click through all three variants across every content area (home, reviews —
list/add/edit for at least a couple of the 5 types, vinyl, workouts, listening, projects). Decide
which variant you want to keep.

## Done when

- You've reviewed all three variants live.
- You've recorded your decision by appending a `## Decision` section to the END of
  `.harness/designs/T042-ui-rewrite-briefs.md`, stating which variant number won (1, 2, or 3) and
  a brief one-line reason — this is the durable record the next task (T065) reads to know which
  variant to promote and which two to delete.
- This task is marked done via `.harness/mark-done.sh T064` (or the dashboard's "Mark done"
  button) once the decision is recorded.
