## Overview
The owner confirmed this migration should go beyond parity with Google Books and pull richer per-book data Hardcover's GraphQL schema likely exposes but the search endpoint alone doesn't return ŌĆö a synopsis/description, series/reading-order metadata, a canonical Hardcover slug/URL, and Hardcover's own community rating. This matters especially for this collection given how many entries are series books (e.g. Horus Heresy).
