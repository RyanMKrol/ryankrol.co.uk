# T077 — Remove Open Library from book search (API + toggle UI)

## Do

Consolidate book search onto Google Books only, removing Open Library entirely:

- **`src/pages/api/books/search.js`**: remove the `provider` query-param switch and the
  `searchOpenLibrary` function and its `mapBookResult` import from `../../../lib/openlibrary`.
  The route should always search Google Books (the logic currently in `searchGoogleBooks`),
  regardless of any `provider` param passed (or simply stop reading `provider` from the query
  string at all — your call, but the route must no longer be able to search Open Library under
  any input).

  `PROVIDERS` array with `{key:'openlibrary', label:'Search Open Library'}` /
  `{key:'googlebooks', label:'Search Google Books'}`, and the buttons/state that let the user pick
  between them). Replace with a single "Search" action that always searches Google Books — no
  toggle, no choice presented, since there is now only one provider.

Do NOT touch `src/lib/openlibrary.js` or its test file in this task — those still exist on disk
and are handled by the dependent T078 (deleting them safely, once nothing references them, is a
separate step so this task's diff stays focused on the API/UI change alone).
