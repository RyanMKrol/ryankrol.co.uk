# T003 — Provision the PerfumeRatings table (human step)

## Do

Run the create-table script from T002 to provision the `PerfumeRatings` table in
DynamoDB. This requires the owner's AWS credentials, so it cannot be done by the
autonomous loop.

```
npm run perfume:create-table
```

## Done when

- `npm run perfume:create-table` has completed successfully against the owner's AWS
  account, and the `PerfumeRatings` table shows status ACTIVE in DynamoDB
  (region `us-east-2`) with a single String partition key `id`.
