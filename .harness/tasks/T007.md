# T007 — Run the perfume seed import (human step)

## Do

Run the seed-import script from T006 to load the fragrances into the `PerfumeRatings`
table. Requires the owner's AWS credentials and the local gitignored
`.harness/perfume-seed-data.tsv`, so the autonomous loop cannot do it.

```
npm run perfume:seed
```

## Done when

- `npm run perfume:seed` has completed successfully, the `PerfumeRatings` table contains
  the imported fragrances, and `/reviews/perfumes` shows them in the list.
