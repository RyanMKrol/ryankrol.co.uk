# T180 — Perfume card variant 1: Stat Block (data-dense)

## What was built
- `src/components/perfumeVariants/Variant1StatBlock.js`: standalone, default-exported
  component accepting `{ item }` (raw `PerfumeRatings` record). Renders title, designer,
  a `type` `Badge` (accentColor=`var(--accent-perfumes)`), rating via `PipMeter`
  (`readOnly`, value=`item.rating`) plus raw `/10` number, a compact stat block with
  labelled longevity (`Longevity: n/8`) and projection (`Projection: n/4`) rows, season
  chips, and application spots as `n× Spot` list. Optional fields (`longevity`,
  `projection`, `seasons`, `applicationSpots`, `description`) only render when present
  on the record. Fragrantica link opens in a new tab with `rel="noopener noreferrer"`.
- `src/styles/globals.css`: added new `.perfume-v1-*` rules at the end of the file —
  only additions, no existing rule touched.
