# T382 — Preserve intentional blank lines in the Markdown block-mode renderer

## Summary

CommonMark (via react-markdown/remark-gfm) collapses any run of consecutive blank lines
down to a single paragraph break, so a blank line a user types between two paragraphs in
the `/tom` "Top of mind" editor had no visible effect — neither in the live preview nor
on the rendered homepage. Added a pure `preserveBlankLines(src)` helper to
`src/components/Markdown.js` that expands each run of N>=2 consecutive blank lines into
the normal paragraph break plus (N-1) standalone "gap paragraphs" — each a lone
non-breaking space (` `) delimited by blank lines, so react-markdown renders it as
its own `<p>` and the extra vertical space actually paints.

## Changes

- **src/components/Markdown.js** — added `preserveBlankLines(src)` (exported for
  testing): splits on `\n`, walks runs of blank (whitespace-only) lines, and for any run
  of length N inserts N-1 extra ` `-only lines each flanked by blank lines. A run of
  exactly 1 blank line (the normal single-paragraph-break case) is left untouched.
  Non-string input passes through unchanged. Called on `children` only in the block-mode
  `<ReactMarkdown remarkPlugins={[remarkGfm]}>` branch — the `inline` branch (used by
  `ReviewCard` for album track highlights) is untouched, still passes `children` straight
  through.
- **src/components/Markdown.test.js** — added tests: (1) one extra blank line between two
  paragraphs (`para one\n\n\npara two`, i.e. 2 consecutive blank lines in source) yields 3
  `<p>` elements — the two content paragraphs plus a ` ` gap paragraph between them;
  (2) two extra blank lines (3 consecutive blank lines) yields 4 `<p>` elements — roughly
  double the gap of case (1); (3) a normal single blank line (`para one\n\npara two`)
  still yields exactly 2 `<p>` elements, unchanged; (4) the same 2-blank-line input in
  `inline` mode emits zero `<p>` elements and no ` ` in the text content — inline
  mode is unaffected. Existing bold/italic/list/link tests unchanged and still pass.
