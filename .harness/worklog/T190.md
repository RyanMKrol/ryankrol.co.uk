# T190: fix perfumes search/sort spacing

## What

`.search-container` div (`margin-bottom: 2rem` only, no flex/gap), causing them to sit
directly on top of each other. Every other review-list page (movies/tv/books/albums/vinyl)
instead nests them in `.collection-review-controls` (flex, gap:1rem) as a sibling of
`.collection-review-title-group` inside `.collection-review-header`.

sibling of `collection-review-title-group` inside `collection-review-header`, matching
`src/pages/vinyl/index.js`. Moved the `{searchTerm && <div className="search-results-count">}`
block out to be a top-level sibling directly after `collection-review-header` (before
`perfume-card-grid`). Deleted the now-empty `search-container` wrapper. No CSS changes —
`.collection-review-controls` etc. already exist in globals.css. Pure JSX relocation, same props.
