# T056 — Variant 2 — workouts + exercises pages

## What was built

- `src/components/v2/V2WorkoutList.js` — workout list (hero card for latest workout +
  masonry grid of the rest), push/pull/legs filter, pagination. Reuses
  `filterWorkouts`/`paginateWorkouts` from `src/lib/workoutPagination.js`.
- `src/pages/v2/workouts/index.js` — renders `V2WorkoutList`.
- `src/pages/v2/workouts/[id].js` — workout detail as a magazine "article" (centred
  680px column, large serif headline, byline-style meta line, drop-cap on the first
  exercise section, sets rendered as a card grid). Fetches `GET /api/workouts/[id]`
  and `GET /api/workouts/[id]/exercises`, unchanged.
- `src/pages/v2/exercises/[exerciseName].js` — exercise progress page, same article
  shell, reusing the existing `ExerciseProgressCharts`/`CardioProgressCharts`/
  `DateRangeFilter` components and `filterByDateRange` lib helper (imported, not
  modified) so chart behaviour matches production. Fetches
  `GET /api/exercises/history/[exerciseName]`, unchanged.

All three pages use the Variant 2 `V2Layout` masthead/colophon wrapper from T044 —
no second layout was built.
