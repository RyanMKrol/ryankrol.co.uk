# T051 — Variant 1 — listening + projects pages

## What was built

- `src/pages/v1/listening/index.js` — dense table of Last.fm top albums (3-month),
  fetched unchanged from `GET /api/lastfm/top-albums?period=3month&limit=50`. Columns:
  rank, album, artist, plays (right-aligned, tabular nums), sortable headers, row click
  opens the shared right-hand detail panel (cover art, artist, rank, plays, last.fm link).
- `src/pages/v1/projects/index.js` — dense table of GitHub repos, fetched unchanged from
  `GET /api/github/repos`. Columns: repo (+ private badge), language, description, stars,
  forks, updated. Detail panel shows full description, topics, and a GitHub link.

Both pages reuse `V1Layout` (sidebar/breadcrumb chrome) from T043 and follow the same
filter/sort/paginate/detail-panel pattern already established in
`V1ReviewTable.js`/`v1/vinyl/index.js`/`v1/workouts/index.js` — dense `<table>`, 34px rows,
zebra striping, sortable column headers, `1–50 of N` pager, no card padding, per Variant 1's
brief.
