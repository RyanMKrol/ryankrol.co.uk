# T192 — exercise-name preview on workouts list

## What changed
- `src/pages/workouts/index.js`: added `getExercisePreview(exercises, maxShown = 3)` helper
  (reads `.title` off each item in `workout.exercises`, already present client-side from
  `GET /api/workouts?mode=all` — no new fetch). Renders a `.workout-session-exercise-preview`
  `<p>` between the datetime line and the stats row, joined with ` · `, with a trailing
  `+N more` when there are more than 3 exercises. Renders nothing (not an empty tag) when a
  workout has 0 exercises or all entries lack a `.title`.
- `src/styles/globals.css`: added `.workout-session-exercise-preview` next to the existing
  `.workout-session-datetime` rule (T159 block) — same mono font / `--color-ink-mute` / 0.85rem,
  plus `overflow: hidden; text-overflow: ellipsis; white-space: nowrap;` so a long exercise list
  can't break the card border.
