# T223 — masonry-style reflow for books/reviews spine-cover list via CSS multi-column

## What changed
`src/styles/globals.css`:
- `.spine-cover-list`: replaced `display: grid; grid-template-columns: repeat(2, 1fr); gap: 1.5rem;`
  with `column-count: 2; column-gap: 1.5rem;`.
- `@media (max-width: 700px) { .spine-cover-list }`: `grid-template-columns: 1fr` → `column-count: 1;`.
- `.spine-cover-card`: added `break-inside: avoid;` and `margin-bottom: 1.5rem;` (replaces the
  vertical spacing the old grid `gap` provided, since multi-column has no row-gap equivalent).

Mirrors the exact pattern shipped for `.projects-grid`/`.project-card` in commit 796102b (T210).
