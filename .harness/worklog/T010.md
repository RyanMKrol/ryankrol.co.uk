- `src/components/DateRangeFilter.js`: presentational segmented-control component using inline styles (CSS custom props for theming), no external CSS file dependency.

## Key decision: UTC month math

`setUTCMonth`/`getUTCMonth` used instead of `setMonth`/`getMonth` to avoid local-timezone drift when computing the cutoff date. Without this, the "exactly 3 months ago" boundary test failed on machines with a western UTC offset because the local-time month arithmetic shifted the cutoff past the item's UTC timestamp.
