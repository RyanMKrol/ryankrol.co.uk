# T161 ‚Äî Exercise detail page restyle

## What changed

- `src/pages/exercises/[exerciseName].js`: rebuilt using `collection-back-link`, `PillGroup`
  (mapped from `DATE_RANGES`, same `filterByDateRange` behavior/data-fetching untouched),
  `StatBlock` x5 (sessions, all-time 1RM [accent `--accent-movies` coral], max weight, total
  volume, 1RM progress % [accent `--accent-books` teal/green]) for strength exercises, and an
  equivalent neutral 5-stat block for cardio exercises. Session history is now a real
  `<table className="exercise-sessions-table">` with Date/Split ¬∑ Volume ¬∑ Max Wt ¬∑ Est. 1RM
  columns (split color reused from the workout-detail page's `SPLIT_COLORS` convention).
- `src/components/ExerciseProgressCharts.js`: "Estimated 1RM" is now a coral area chart with a
  canvas gradient fill (`gradientFill()` helper derives the gradient from the theme's
  `chartCoral` hex ‚Äî no colors hardcoded outside the hook), "Session volume" is now a `Bar`
  chart (indigo, last bar overridden to grape via a per-index `backgroundColor` array), "Max
  weight" is a marigold area chart. The insights panel switched from the generic `.insights-panel`
  class to the new dark `.exercise-insights-panel`.
- `src/components/CardioProgressCharts.js`: only the insights-panel class swapped to
  `.exercise-insights-panel` for a consistent dark panel; its own chart colors (primary/secondary/
  tertiary) are untouched ‚Äî the design brief only specifies coral/indigo+grape/marigold for the
  strength charts this page renders.
- `src/hooks/useChartTheme.js`: added `chartCoral(Bg)`, `chartIndigo(Bg)`, `chartGrape`,
  `chartMarigold(Bg)` as new, additive keys (both the SSR fallback object and the
  `getComputedStyle`-backed browser path). No existing key was renamed, removed, or had its
  meaning changed, so `src/components/ProgrammeOverviewCharts.js` (the unrelated `/programmes`
  feature, out of scope here) keeps consuming `chartPrimary`/`chartSecondary`/`chartTertiary`/
  `chartCardDefault` exactly as before ‚Äî verified by grep, not by opening the file.
- `src/styles/globals.css`: added matching `--color-chart-coral/-indigo/-grape/-marigold(-bg)`
  overrides to the `html.matrix-active` block (mapped to shades of matrix green, matching the
  existing pattern for `--color-chart-primary/secondary/tertiary` so charts stay monochrome
  green under the Matrix theme). Added `.exercise-detail-stats` (5-col stat grid, 2-col on
  mobile), `.exercise-insights-panel(-title)` (dark ink panel, light text ‚Äî same pattern as the
  home page's `.home-gym-panel`), and `.exercise-sessions-table` (Collection-styled table).
