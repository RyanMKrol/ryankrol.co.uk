# T072 — Show the full repo list on /projects (remove the top-20 cap)

## What changed
- `src/pages/api/github/repos.js`: replaced the single unpaginated GitHub API fetch with a loop
  that requests `per_page=100` pages, following `page=2, 3, ...` until a page returns fewer than
  100 results, concatenating all pages before filtering/sorting.
- Removed the `.slice(0, 20)` cap after the sort.
- Filter (`!repo.fork && !repo.archived && repo.pushed_at`) and sort (most-recent-push first) are
  unchanged. Response shape (`{ repos, total, username }`) and `withApiCache` wrapper unchanged.
