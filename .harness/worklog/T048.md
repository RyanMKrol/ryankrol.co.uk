# T048 — Variant 1 reviews edit pages

## Built

- `src/components/v1/V1EditList.js` — generic edit-list table (dense grid, zebra rows, click-through
  to detail) shared by all 5 review types' edit list pages.
- `src/components/v1/V1FormShell.js` — added `V1FormActions` and `V1FormDanger` exports (a red-accent
  button matching the existing dense form styling) for the save/delete pair on edit forms.
- `src/pages/v1/reviews/{movies,tv,books,albums,perfumes}/edit.js` — list pages using `V1EditList`,
  reusing the unchanged `GET /api/reviews/<type>` routes and T043's `V1Layout` breadcrumb/sidebar
  chrome.
- `src/pages/v1/reviews/{movies,tv,books,albums,perfumes}/edit/[id].js` — detail/edit forms built on
  `V1FormShell`, mirroring the production `edit/[id].js` field sets and key-encoding conventions
  (movies/tv: title; books: title|author; albums: title|artist; perfumes: id) and calling the
  existing unchanged `update`/`delete` API routes.
