# T224 — masonry-style reflow for poster-banner grid via CSS multi-column

## Do

Converted `.poster-banner-grid` from CSS Grid to CSS multi-column, mirroring the exact
pattern shipped for `.projects-grid`/`.project-card` in commit 796102b (T210):

- `.poster-banner-grid`: `display: grid; grid-template-columns: repeat(3, 1fr); gap: 1.5rem;`
  → `column-count: 3; column-gap: 1.5rem;`
- `@media (max-width: 900px) { .poster-banner-grid }`: `grid-template-columns: 1fr` →
  `column-count: 1;`
- `.poster-banner-card`: added `break-inside: avoid;` and `margin-bottom: 1.5rem;` to
  replace the row-spacing the old grid `gap` provided.

Pure CSS change — no JS/markup touched. This class is shared by both
`src/pages/reviews/movies/index.js` and `src/pages/reviews/tv/index.js` via the
`poster-banner` styleVariant, so this single change fixes both pages.
