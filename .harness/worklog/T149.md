# T149 — Rebuild the home page to the Collection design

## What changed
- `src/pages/index.js`: fully rebuilt on the Collection design (`renders/1a.png` desktop /
  `4a.png` mobile — `1b.png` intentionally NOT referenced, it's the rejected terminal concept).
  Fetches `/api/reviews/{movies,tv,books,albums}`, `/api/vinyl`, and `/api/workouts/stats`
  client-side in one `Promise.all`, no new API routes added. Sections:
  - Hero: accent badge, "Howdy." display heading, tagline, right-aligned mono meta line
    (logging since 2019 / real derived "N things rated" / "updated today").
  - 6 `StatBlock`s (movies/tv/books/albums/vinyl/workouts) with real counts, each accent-filled
    per section token.
  - "The collection wall": 9-col `CoverTile` grid merging top-rated movies (TMDB poster via
    `tmdbPosterUrl`), top-rated albums (Last.fm `thumbnail`), and vinyl (`thumbnail`) — falls
    back to `CoverTile`'s gradient placeholder when no cover exists (visible for one album in
    the desktop screenshot).
  - "Latest takes": 3 most-recent reviews across movies/tv/books/albums, sorted by parsed
    `DD-MM-YYYY` date, with a small cover thumb + star rating + snippet.
  - Gym rail: session count + "best session vol" (max `totalVolume` across
    `workoutStats.recentActivity`, since the existing `getWorkoutStats()` doesn't compute a
    cross-exercise best-1RM and adding that would need a new/heavier endpoint — out of this
    task's scope) + a CSS bar sparkline over the same recent-activity volumes.
  - Vinyl shelf: first 4 records (title + artist) from `/api/vinyl`.
  - Mobile (`<=768px`): stats collapse to 3-col, "the collection wall" is hidden and replaced by
    3 quick-link pill rows (`~/vinyl`, `~/listening`, `~/projects`).
  - `Header` (from T146) is rendered directly on the page, matching how every other rebuilt page
    already does it (`_app.js` doesn't render `Header` globally — confirmed while re-reading the
    codebase for this task).
- `src/styles/globals.css`: added a new "Home page (Collection redesign, T149)" block — hero,
  stats grid, wall grid, quick-links, latest-takes cards, gym panel + sparkline, shelf panel, and
  the `max-width: 768px` collapse rules. Left the old pre-redesign `.home-grid*` / `.terminal-prompt*`
  rules in place (unused now, but out of scope to prune here — no other page references them either).
