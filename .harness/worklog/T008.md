# T008 worklog

## What was built

Added `src/lib/programmeStats.js` with two pure helpers:

- `classifyProgramme(workout)` — case-insensitive substring match of `push`, `pull`,
  `legs` against `workout.title`; returns `null` for anything else.
- `aggregateProgramme(workouts, programme)` — filters to matching programme, sorts
  date-ascending by `start_time`/`workoutDate`, and returns:
  - `sessions`: `[{ date, volume, workingSets }]`
  - `frequencyByMonth`: `[{ month: 'YYYY-MM', count }]`
  - `totals`: `{ workouts, totalVolume, avgVolumePerWorkout }` (avg rounded via `Math.round`)

Also added `src/lib/programmeStats.test.js` with 10 hermetic tests covering classification,
programme filtering, date-sort, field mapping, month-frequency, totals math, and empty input.
