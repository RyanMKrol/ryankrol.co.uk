# T044 — Variant 2 shared layout/nav shell + home page

## What was built

- `src/components/v2/V2Layout.js` — shared chrome for the "Editorial Card Magazine"
  variant (T042's Variant 2 brief): a centered masthead with a large serif wordmark
  and a small-caps section row beneath it (single chrome, no sidebar), and a
  three-column "colophon" footer band (about/tagline, repeated section links, a
  reused `NowPlaying` widget). Section row scrolls horizontally on narrow widths
  instead of collapsing into a hamburger drawer, per the brief. Exports
  `V2_SECTIONS` (the 9 section links: movies/tv/books/albums/perfumes/vinyl/
  workouts/listening/projects, all under `/v2/**`).
- `src/pages/v2/index.js` — home page using `V2Layout`: one large hero card (serif
  headline, kicker, blurb) for a "featured" section, then the remaining four
  sections in a CSS-columns masonry grid with alternating card heights, per the
  brief's asymmetric/masonry list-view pattern.
- Own look per the brief (serif/sans font pairing, cream/ink palette, double-rule
  borders) — does not reuse `MatrixLayout` or the production theme's CSS variables.
