# T215 ‚Äî Sort control flip-direction fix

## What changed
Replaced the single-select `PillGroup`/`SORT_OPTIONS` sort control with the existing
`SortButtons`/`SORT_FIELDS` component on all four review list pages (movies, tv, books,
albums). `SortButtons.handleClick` already implements flip-on-reclick (default ‚Üí flipped ‚Üí
default) and activate-at-default-on-switch, so re-clicking the active pill now actually
changes order instead of being a no-op.

Each page's sort `useEffect` now branches explicitly on `title`, `title-desc`, `score`,
`score-asc`, `date-asc`, with a trailing `else` covering the initial `date` state (and any
legacy/unrecognized value) as newest-first, matching pre-fix behavior.
