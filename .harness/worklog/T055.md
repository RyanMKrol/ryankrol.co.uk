# T055 — Variant 2 — vinyl pages

## What was built

- `src/pages/v2/vinyl/index.js` — vinyl collection list, styled per Variant 2's brief
  (Editorial Card Magazine): hero card for a featured record, masonry grid of the rest,
  Georgia/serif titles over sans body, cover art from `thumbnail` where present. Uses the
  shared `V2Layout` from T044.
- `src/pages/v2/vinyl/add.js` — add-vinyl form, built on the shared `V2ArticleForm` article
  shell (underline-only inputs, centered column) also from T044. Reuses the existing
  `LastfmAlbumSearch` component unchanged.

Both pages fetch from the existing, unmodified API routes (`GET /api/vinyl`,
`POST /api/vinyl/add`) — no file under `src/pages/api/**` was touched.

## Verification

- `npm run lint` — 0 errors (2 pre-existing warnings unrelated to this change).
- `npm test` — 137/137 passing.
- `npm run build` — succeeds; `/v2/vinyl` and `/v2/vinyl/add` both listed as static routes.
- `npm run dev` + curl: `GET /v2/vinyl` → 200, `GET /v2/vinyl/add` → 200, no errors in the
  dev server log.

## Scope

Only `src/pages/v2/vinyl/index.js` and `src/pages/v2/vinyl/add.js` were added. No file
outside `src/pages/v2/**` / `src/components/v2/**` was modified.
