# T145 — CoverTile component

## What was built
- `src/components/CoverTile.js`: generic square/near-square cover tile.
  - `gradientForKey(key)` derives a stable index into a fixed set of 9 two-color diagonal
    gradient pairs (coral, indigo, teal, marigold, pink, grape, green, plus two neutrals) via a
    simple string hash — same key always yields the same gradient.
  - Renders an `<img>` filling the tile when `imageUrl` is passed, otherwise the gradient
    placeholder background.
  - `aspectRatio` prop (default `1 / 1`) so vinyl/album/book callers can override if needed later.
  - Caption block below the tile: bold `title` + optional muted `subtitle` (generic name, not
    "artist", so it also reads for book authors etc).
- `src/styles/globals.css`: added `.cover-tile-wrap/.cover-tile/.cover-tile-image/.cover-tile-caption/
  .cover-tile-title/.cover-tile-subtitle` rules using the existing `--radius-cover` / `--shadow-cover`
  tokens (no new tokens needed — T142 already added them).
