# T225 — masonry-style reflow for workout session grid via CSS multi-column

## What changed
`src/styles/globals.css`:
- `.workout-session-grid`: replaced `display: grid; grid-template-columns: repeat(2, 1fr); gap: 1.25rem;`
  with `column-count: 2; column-gap: 1.25rem;` (kept `margin-bottom: 2rem;`).
- `@media (max-width: 900px) .workout-session-grid`: `grid-template-columns: 1fr` → `column-count: 1;`.
- `.workout-session-card`: added `break-inside: avoid;` and `margin-bottom: 1.25rem;` to replace the
  row-spacing previously provided by grid `gap`.

Exact mirror of the `.projects-grid`/`.project-card` pattern from commit 796102b (T210).
