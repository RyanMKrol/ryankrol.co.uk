# T156 — Rebuild vinyl page to Collection design

## What changed
- `src/pages/vinyl/index.js`: rebuilt header (title + "N records on the shelf · sorted by
  artist" meta), added `SearchInput` (filters by title/artist) and a `PillGroup` covers/list
  toggle (`--accent-vinyl`). Kept the existing artist-surname-first-letter grouping logic
  (`groupByLetter`/`getArtistForSorting`, unchanged behaviour) but changed the per-letter
  header to the new `vinyl-letter-header` treatment (big accent letter + hairline rule +
  "N records" count). Covers view renders a `CoverTile` grid per letter using `record.thumbnail`
  as `imageUrl` (falls back to `CoverTile`'s gradient placeholder when absent, e.g. the ~2/190
  records without a cover). List view is a simple title/artist row table (no explicit mock
  existed for it).
- `src/styles/globals.css`: added `.vinyl-letter-header*` (accent letter, hairline, count),
  `.vinyl-cover-grid` (7-col desktop, 3-col under `max-width: 700px`), and `.vinyl-list-row*`
  for the list view. Left the old `.vinyl-section-header`/`.vinyl-record` rules in place
  (harmless, no longer referenced by this page but scope only allows touching these two files).
