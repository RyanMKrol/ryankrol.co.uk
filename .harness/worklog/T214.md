# T214: workouts session card volume — tonnes to kg

## What changed
- `src/pages/workouts/index.js`: renamed `formatVolumeTonnes` → exported `formatVolumeKg`,
  now `Math.round(volumeKg)` instead of `(volumeKg/1000).toFixed(1)`.
- Call site (session card volume, inside the workouts-session-card-layout markup from T213)
  updated to `{formatVolumeKg(getTotalVolume(exercises))}kg` (was `...}t`).
- Added `src/tests/workoutsIndex.test.js` covering exact/round-down/round-up kg cases
  (8600 → "8600", 8600.4 → "8600", 8600.6 → "8601"). Placed outside `src/pages/workouts/`
  rather than colocated, because a `*.test.js` file under `src/pages/**` is picked up by
  Next.js as a page route (pageExtensions includes `js`) and broke `next build` with
  "Failed to collect page data for /workouts/index.test" — no existing non-API page test
  precedent in this repo to follow instead.
