- `src/components/PerfumeCharacteristics.js`: exported `PROJECTION_LABELS` (was private) so it can
  be reused outside the form sliders.
- `src/components/perfumeVariants/Variant6Hybrid.js`: wrapped the `.perfume-v4-scale-track` divs
  for both Longevity and Projection in the shared `Tooltip` component.
  - Longevity label reuses the same copy as `LongevitySlider`: `${value >= 8 ? '8+' : value} hours`

    `ProjectionSlider`).
  - Wrapped the whole track (not just the fill), so hovering/focusing anywhere along the bar's
    width triggers the tooltip, per spec.
