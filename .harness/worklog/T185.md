# T185 — Perfume card variants side-by-side preview page

## What was built

`src/pages/dev/perfume-variants.js` — a dev-only, unlinked page that fetches real perfume
records from the existing `GET /api/reviews/perfumes` endpoint (same client-side `useEffect`
fetch pattern as `src/pages/reviews/perfumes/index.js`) and renders the first 8 records through
all five variant components (`Variant1StatBlock` ... `Variant5TableRow` from
`src/components/perfumeVariants/*`) as five vertically stacked, labelled sections, using the
same subset of records across all five for apples-to-apples comparison. Grid layout for the
four card/tile variants, a flex list for the table-row variant. Page-scoped inline styles only;
no changes to `globals.css` or any API route.
