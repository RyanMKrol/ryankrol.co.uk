# T300: Add hover tooltips to /projects card star and commit-count badges

## What changed
- `src/pages/projects/index.js`: imported the shared `Tooltip` component and wrapped the
  `.project-card-stars` and `.project-card-commits-badge` spans in `.project-card-top-right`
  with it, each labelled with the spelled-out count noun (`"N star(s)"` / `"N commit(s)"`).

  `Tooltip`+span for the commit badge.

  top of each card — the default top-positioned bubble would clip against the card/page edge
  for first-row cards.
