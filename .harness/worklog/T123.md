# T123: stamp editedDate on album review updates

## What changed
- `src/pages/api/reviews/albums/update.js`: added `editedDate = :editedDate` to
  `updateExpressionParts` and `expressionAttributeValues[':editedDate']` computed
  unconditionally via `new Date().toLocaleDateString('en-GB').replace(/\//g, '-')`,
  same convention used in the sibling `add.js` routes and in T120/T121 for
  movies/tv. `date` remains untouched since `UpdateCommand` only writes attributes
  named in the expression.
