# T147: Build shared Footer component

Prior attempt at this task correctly stopped rather than fake success (see the recovery note this
replaces): `src/pages/index.js` wasn't in `T147`'s `scope`, so the loop's structural gate would have
refused the required edit. That's since been fixed (`index.js` added to scope) — this is the fresh
build.

## What changed
- New `src/components/Footer.js`: renders the 5 social links (instagram, facebook, github,
  linkedin, last.fm — same hrefs as the old inline `index.js` footer) plus a
  "© ryan krol · made with too much free time" copyright line, per the `1a.png` render
  (left-aligned links, right-aligned copyright, hairline top border).
- Wired `Footer` into `src/pages/_app.js` inside `MatrixLayout`, alongside `Component`, so it
  renders once per page.
- Removed the inline footer markup from `src/pages/index.js` (was lines ~70-117) now that it lives
  in the shared component.
- `src/styles/globals.css`: replaced `.home-social-link` (home-page-only) with `.site-footer`,
  `.footer-content`, `.footer-social-links`, `.footer-social-link`, `.footer-copyright` — uses
  `var(--color-ink-soft, var(--color-text-muted))` so it falls back gracefully under the Matrix
  theme (which has no `--color-ink-soft` token), `var(--font-mono)`, and the same
  `var(--gutter-mobile)` horizontal padding pattern as `.site-header`.
