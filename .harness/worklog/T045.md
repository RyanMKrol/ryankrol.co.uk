# T045 — Variant 3 — shared layout/nav shell + home page

## What was built

- `src/components/v3/V3Layout.js` — shared chrome per the Variant 3 brief ("Minimal
  Linear Feed"): a single slim 40px sticky utility bar with a wordmark/home link on
  the left and a `Jump to ▾` `<details>` dropdown on the right listing the five
  content sections (`V3_SECTIONS`, all pointing at `/v3/**` routes). No sidebar, no
  masthead, no footer — content ends with an "— end of feed —" marker on the page
  itself, not in the layout.
- `src/components/v3/V3TimelineEntry.js` — reusable timeline row: fixed-width date
  gutter on the left, `[type]` inline label + summary line, and a `+`/`−` disclosure
  toggle that expands the entry in place (no navigation) to reveal detail content —
  the shared building block later review/vinyl/workout list tasks (T046+) can reuse
  for the "list view = timeline, detail = inline expand" pattern from the brief.
- `src/pages/v3/index.js` — home page rendering `V3Layout` with an inline
  `sorted by date, newest first — change` control, links to the five section routes,
  and a mock interleaved timeline (movie/vinyl/workout/album/tv mock entries) using
  `V3TimelineEntry` to demonstrate the expand-in-place pattern. Section routes don't
  exist yet (land in T046–T063) so those links 404 for now, per the task spec.
