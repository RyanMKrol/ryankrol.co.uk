# T283 — Replace 'consider' checkboxes with an ownership picker

## What changed
- `src/components/PerfumeCharacteristics.js`: added `OwnershipPicker({ value, onChange })`,
  following the existing `LongevitySlider`/`ProjectionSlider`/`SeasonsCheckboxes` pattern —
  wraps `PillGroup` with the `Sample` / `Travel size` / `Full bottle` options.
- `src/pages/reviews/perfumes/add.js`: `formData.ownership` defaults to `'Sample'` (replacing
  `considerTravelSize`/`considerFullBottle` booleans). Checkbox pair replaced with
  `<OwnershipPicker value={formData.ownership} onChange={(ownership) => setFormData({ ...formData, ownership })} />`
  in the same position (between Fragrantica URL and `LongevitySlider`).
- `src/pages/reviews/perfumes/edit/[id].js`: same default + same picker wiring; hydration from
  fetched data now does `ownership: perfume.ownership || 'Sample'` so pre-migration rows (no
  `ownership` field yet) fall back to `'Sample'` active rather than nothing selected.
