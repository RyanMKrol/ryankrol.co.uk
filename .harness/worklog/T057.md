# T057 — Variant 2 — listening + projects pages

## What was built

- `src/pages/v2/listening/index.js` — Last.fm top-albums (3 month) page, fetching from
  the existing `/api/lastfm/top-albums?period=3month&limit=50` route (unchanged). Reuses
  `V2Layout`. Follows Variant 2's brief: hero card for the #1 album ("On repeat"), then
  the rest in a masonry grid, each card showing rank, title, artist, playcount and a
  popularity bar relative to the max playcount.
- `src/pages/v2/projects/index.js` — GitHub projects page, fetching from the existing
  `/api/github/repos` route (unchanged). Reuses `V2Layout`. Hero card for the most
  recently pushed repo, then the rest in a masonry grid with language/stars/forks/updated
  metadata and topic tags, matching the production page's data but restyled per the
  editorial-magazine brief (serif headline type, small-caps kicker/meta, uneven card
  heights).

Both pages match the existing `V2Vinyl` page's structure/patterns (loading/error/empty
states, hero + masonry, inline `<style jsx>`) for consistency with the rest of Variant 2.
