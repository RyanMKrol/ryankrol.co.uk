# T152 — spine-cover ReviewCard variant + books list rebuild

## What changed
- `src/components/ReviewCard.js`: added a `spine-cover` `styleVariant`, rendered via a small
  internal `SpineCoverCard` sub-component (mirrors the existing `poster-banner` branch pattern).
  Layout: gradient cover tile (88x128, reuses `gradientForKey` from `CoverTile.js`) on the left;
  title + read-only `StarRating` on one line, "Author · Npp · Year" meta line
  (from `item.author` / `item.pageCount` / `item.firstPublishedYear`, filtered for missing
  fields), then an italic review-text snippet.
- `src/pages/reviews/books/index.js`: rebuilt to match the T150 movies page pattern —
  `collection-review-header` with rated/avg/this-year summary, `SearchInput` + `PillGroup`
  sort controls (date/title/score) — but rendering a single-column `.spine-cover-list` of
  horizontal cards instead of a grid. No pagination (deferred to T155).
- `src/styles/globals.css`: added `.spine-cover-*` rules under a new "Spine Cover Review Card
  (T152)" section, following the existing poster-banner section's conventions (border-bottom
  row separators instead of individual card borders, since cards are full-width rows).
