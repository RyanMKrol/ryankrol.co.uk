# T184 — Perfume card variant 5 — Minimal Table Row

## What was built
- `src/components/perfumeVariants/Variant5TableRow.js`: standalone presentational
  component rendering a single perfume record as a dense, aligned table row using
  a native `<details>`/`<summary>` element.
  - Summary row (CSS grid, fixed/fr columns): title, designer, type (badge), rating
    (compact `x/10`), longevity (`x/8` or `—`), projection (`x/4` or `—`), seasons
    (comma-joined or `—`).
  - Expandable disclosure (collapsed by default): description, applicationSpots list,
    Fragrantica link — none of these appear in the main row.
- `src/styles/globals.css`: added new `.perfume-v5-*` rules only, appended after the
  existing `.perfume-v4-*` block. No existing rule was modified or removed.
