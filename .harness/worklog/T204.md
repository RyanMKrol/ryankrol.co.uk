# T204 — "each" qualifier for paired application spots at 1 spray

## What changed
- `src/components/perfumeVariants/Variant6Hybrid.js`: added exported
  `PAIRED_APPLICATION_SPOTS` (`Wrists`, `Elbows`, `Clavicles`, `Behind ears`) and exported
  pure `formatApplicationSpotLine({ spot, sprays })`. The `<li>` in the applicationSpots
  map now calls this helper instead of the inline ternary. `sprays === 1` on a paired spot
  renders `"1 spray each — <Spot>"`; on `Beard`/`Back of neck`/`Clothes` it stays
  `"1 spray — <Spot>"`; any `sprays !== 1` is unchanged (`"<N> sprays — <Spot>"`).
- Added `src/components/perfumeVariants/Variant6Hybrid.test.js`, importing only the named
  `formatApplicationSpotLine` export, covering all 7 `VALID_APPLICATION_SPOTS` values at
  `sprays === 1` plus a `sprays: 2` case.
