# T027 worklog — Replace review sort dropdown with segmented sort buttons

## What was built

- Created `src/components/SortButtons.js`: a reusable prop-driven segmented button component.
  - Accepts `fields` (array of `{ key, label, defaultValue, flippedValue, defaultArrow, flippedArrow }`), `sortBy`, and `onChange`.
  - Clicking inactive field → activates at its default direction.
  - Clicking active field → flips direction. No off/disabled state.

  - `movies`, `tv`, `books` → 3 buttons: Date / Title / Score

- Updated `src/styles/globals.css`:
  - Replaced `.sort-container`, `.sort-label`, `.sort-select`, `.sort-select:focus` with `.sort-buttons-container` and `.sort-arrow`.
