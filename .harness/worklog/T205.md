# T205 — Perfume spray-spot circle redesign

## What changed
- `src/components/PerfumeCharacteristics.js`: `SpraySpotCounter` now renders a fixed
  4.5rem circle (`.perfume-spray-spot-circle`) containing only the nose emoji + poof
  animation + spray count, with the spot label moved to a sibling
  `.perfume-spray-spot-label` below the circle. Removed all inline `style={{...}}`
  objects and the component's `<style jsx>` block from this widget.
- `ApplicationSpotsSprayer`'s row container now uses `className="perfume-spray-spots"`
  instead of an inline style object.
- `src/styles/globals.css`: added `.perfume-spray-spots`, `.perfume-spray-spot`,
  `.perfume-spray-spot-circle` (+ `.is-active`), `.perfume-spray-spot-emoji`
  (+ `.is-spraying`), `.perfume-spray-spot-poof`, `.perfume-spray-spot-count`
  (+ `.is-active`), `.perfume-spray-spot-label`, and moved the `spray-poof` /
  `spray-squeeze` `@keyframes` out of the component into this section (grouped after
  `.perfume-v4-fragrantica-link`).
- `LongevitySlider`, `ProjectionSlider`, `SeasonsCheckboxes` untouched, per spec.
- `applicationSpotCounts` / `countsToApplicationSpots` untouched — no data-shape changes.
