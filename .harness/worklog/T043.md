# T043 — Variant 1 shared layout/nav shell + home page

Built the Variant 1 ("Dense Data-Table Console") shared chrome and home page, per
`.harness/designs/T042-ui-rewrite-briefs.md` § Variant 1.

## What was built

- `src/components/v1/V1Layout.js` — shared layout wrapper: fixed ~220px left sidebar
  (flat list of section rows with row-count badges, collapses to an icon-only 56px rail
  under 900px), no top nav bar, single-line breadcrumb + inline search input as the
  header, and a fixed status line (`● online · cache: 12m ago`) at the sidebar foot in
  place of a footer. Exports `V1_SECTIONS` (the shared nav list of the five content
  areas, expanded to their per-review-type routes) for reuse by dependent-task pages.
  Styling is self-contained via `styled-jsx` (dark terminal palette, JetBrains Mono,
  single 14px body size, hierarchy via weight/color only) — does not reuse
  `MatrixLayout` or the production theme's CSS custom properties, per the brief.
- `src/pages/v1/index.js` — home page rendered inside `V1Layout`; the "home page IS the
  sidebar landing state" aggregate table (one row per section, `Type | Section | Items`
  columns, zebra striping, right-aligned numeric column), each row linking into
  `/v1/**`.

Nav links point at `/v1/reviews/movies`, `/v1/reviews/tv`, `/v1/reviews/books`,
`/v1/reviews/albums`, `/v1/vinyl`, `/v1/workouts`, `/v1/listening`, `/v1/projects` —
these 404 today and are expected to land in T046–T063. Row-count badges are hardcoded to
`0` placeholders since no data wiring is in scope for this task.
