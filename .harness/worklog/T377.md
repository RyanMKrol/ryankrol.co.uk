# T377: Toggle formatting for Bold, Italic, and List in MarkdownEditor

## Summary

Implemented toggle functionality for Bold (`**`), Italic (`*`), and List (`- `) formatting in the shared MarkdownEditor component. A second click on already-formatted text now removes the formatting instead of adding another layer.

## Changes Made

### `src/components/MarkdownEditor.js`

- **Added `isWrappedWith(value, start, end, before, after)`** — Pure helper that detects if text is wrapped with formatting markers, checking both:
  - Markers immediately outside the selection (e.g., `**[test]**`)
  - Markers included in the selection (e.g., `[**test**]`)

- **Added `unwrapSelection(value, start, end, before, after)`** — Pure helper that removes formatting markers from text, handling both wrapper positions and returning the unwrapped text with correct cursor positioning.

- **Added `removeLinePrefix(value, start, end, prefix)`** — Pure helper that removes the list prefix (`- `) from a line, mirroring the existing `insertLinePrefix` behavior.

- **Updated `handleToolbarAction()`** to implement toggle logic:
  - **Bold**: Detects if already wrapped in `**`, unwraps if true, otherwise wraps
  - **Italic**: Detects if already wrapped in `*`, unwraps if true; also checks for bold (`**`) first to avoid misidentification
  - **List**: Calls `removeLinePrefix` first, and if it returns null, calls `insertLinePrefix` to add the prefix
  - **Link**: Unchanged (insert-only, no toggle behavior)

### `src/components/MarkdownEditor.test.js`

Added 8 new test cases covering toggle functionality:

1. **toggles Bold off when clicked twice on the same selection** — Verifies bold wrapping then unwrapping yields original text
2. **toggles Italic off when clicked twice on the same selection** — Verifies italic wrapping then unwrapping yields original text
3. **toggles List prefix off when clicked twice on the same line** — Verifies list prefix add then removal yields original text
4. **does not confuse bold (**) with italic (*)** — Verifies clicking italic on bold text wraps both, not removes bold
5. **handles unwrapping when markers are included in selection** — Verifies unwrapping works when `[**test**]` is selected entirely
6. **handles unwrapping when markers are outside selection** — Verifies unwrapping works when `[test]` is selected (markers outside)
7. **preserves focus and selection after toggling off formatting** — Verifies textarea focus and selection are restored correctly
8. **Link button remains insert-only and does not toggle** — Verifies Link button never toggles, only wraps

All 8 new tests pass, and all 6 existing MarkdownEditor tests still pass (unchanged first-click behavior).
