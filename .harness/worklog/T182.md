# T182 — Perfume card variant 3 — Bottle Shelf (visual tile grid)

## What was built

`src/components/perfumeVariants/Variant3ShelfTile.js` — a standalone presentational component
taking `{ item }` (raw `PerfumeRatings` record). Renders a tall (`3 / 4` aspect-ratio) tile
filled with `gradientForKey(item.id || item.title)` (reused from `CoverTile.js`), with the
title and a compact (scaled-down) read-only `PipMeter` overlaid on the tile, plus small
corner chips for `seasons` (one/two-letter abbreviations: W, Sp, Su, A, D, N). Below the tile,
a slim caption strip shows designer, the `type` badge, and a Fragrantica link. Deliberately
omits longevity, projection, applicationSpots, and description per spec — this variant is a
scan-fast visual grid, not a detail card.

New CSS: `.perfume-v3-*` classes appended to the end of `src/styles/globals.css`. No existing
rule was edited or removed (verified via `git diff` — only additions).
