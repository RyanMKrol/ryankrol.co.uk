# T061 — Variant 3 — vinyl pages

## What was built
- `src/components/v3/V3VinylFeed.js` — linear-feed list view for vinyl, following Variant 3's
  brief (date/changelog-style gutter, one flat hr-separated row per record, inline `+` disclosure
  expanding to Last.fm summary notes). Uses the artist's sorting letter (ignoring leading "The")
  as the gutter label since vinyl records have no date field, sorted A–Z by artist then title —
  mirrors production `/vinyl`'s existing grouping logic. Reuses `V3Layout` and `V3TimelineEntry`
  from T045/T060, no new layout wrapper.
- `src/pages/v3/vinyl/index.js` — thin page rendering `V3VinylFeed`.
- `src/pages/v3/vinyl/add.js` — add flow using `V3AddEntry` (pinned top-of-feed entry that
  expands into an inline form, per the brief), reusing the existing `LastfmAlbumSearch`
  component for the optional Last.fm match, posting to the unchanged `/api/vinyl/add`.

Both pages fetch from the existing `/api/vinyl` and `/api/vinyl/add` routes, unchanged.
