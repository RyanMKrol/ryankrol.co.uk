# T021 worklog — Open Library search proxy route + normaliser

## What was built

- `src/lib/openlibrary.js` — pure normaliser library:
  - `mapBookResult(doc)` maps a raw Open Library search doc to the rich field set: `olid`, `title`, `authors`, `firstPublishedYear`, `coverId`, `isbn`, `subjects`, `pageCount`, `publisher`
  - `bookCoverUrl(coverId, size)` builds a `covers.openlibrary.org` URL; returns null for falsy coverId
- `src/lib/openlibrary.test.js` — 12 hermetic unit tests; no live network calls
- `src/pages/api/books/search.js` — GET proxy to Open Library `/search.json`; wraps in `withApiCache(generateCacheKey('book-search', { query }))`; keyless (no env var)
- `CLAUDE.md` — added two rows to the file map for the new route and lib

## Verification

- `npm test -- --testPathPattern=openlibrary`: 12/12 passed
- `npm run build`: succeeded cleanly, `/api/books/search` appears as a Dynamic route
- No live API calls made during verification
