# T294 — set-row PR medal on /workouts/[id]

## What changed
- `src/pages/workouts/[id].js`: in the set-row rendering block, each set now checks
  `set.isWeightPR` / `set.is1RMPR` / `set.isVolumePR` (already present on the exercises
  response, no new fetch) and renders one small `Badge` per true axis, inline next to the
  set's value (weight/reps/etc). Mirrors the medal `Badge` pattern already used for the
  per-exercise preview on `/workouts` (🏅 + visually-hidden descriptive text), but per-axis
  here: "weight PR" / "1RM PR" / "volume PR" markers so a set that hits two axes at once
  shows two distinct badges rather than one ambiguous marker.
- Wrapped the set's value + badges in a flex span so `justify-content: space-between` on
  `.workout-set-row` still applies between the label and this group (unchanged 2-child row
  layout), instead of spreading 3+ children evenly.
