# T148 — Build shared StatBlock component

## Do

Added `src/components/StatBlock.js`: a rounded stat block (`var(--radius-stat)`) with a big
number (`var(--font-display)`, clamps 40px–52px) and a small uppercase label beneath. Two fill
modes via the `accentColor` prop:

- **accent mode** (`accentColor` set): solid fill in the given accent color, white text — for the
  home page's six per-section stat blocks.
- **neutral mode** (`accentColor` omitted): card background + ink border — for bordered stat
  chips such as workout detail's date/duration/volume.

An optional `unit` prop renders a smaller suffix after the number (e.g. `79` + `kg`, matching
`renders/2c.png`'s "ALL-TIME 1RM / 79kg" card).

Added the supporting CSS (`.collection-stat-block`, `.collection-stat-value`,
`.collection-stat-unit`, `.collection-stat-label`) to `src/styles/globals.css`, following the same
plain-className pattern as the existing `Pill` component.
