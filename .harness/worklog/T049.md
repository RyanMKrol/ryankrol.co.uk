# T049 — Variant 1 vinyl pages

## What was built

- `src/pages/v1/vinyl/index.js` — vinyl collection list per Variant 1's "Dense Data-Table
  Console" brief: a zebra-striped `<table>` (Title/Artist/Format/Label columns), sortable by
  clicking Title/Artist headers, row click opens a 420px right-hand slide-in detail panel
  (cover thumbnail, artist, format, label, condition, notes), compact `N–M of T  ‹ ›` pagination
  footer, and a filter input — matching the pattern already established for reviews tables
  (`V1ReviewTable`) but built standalone since vinyl has no rating/date fields to reuse that
  component's column set for.
- `src/pages/v1/vinyl/add.js` — add-vinyl form using the existing `V1FormShell`/`V1FormRow`/
  `V1FormSubmit` primitives from T047/T048 (dense label:value row stack, no card styling),
  reusing the existing `LastfmAlbumSearch` component for optional cover lookup — mirrors
  `src/pages/v1/reviews/albums/add.js` minus rating/highlights fields.
- Both pages use the shared `V1Layout` wrapper from T043 (sidebar nav already lists `/v1/vinyl`)
  and fetch from the unchanged production API routes (`GET /api/vinyl`, `POST /api/vinyl/add`).
